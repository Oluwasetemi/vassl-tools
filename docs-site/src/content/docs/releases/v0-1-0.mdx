This is the initial development release of VASSL. All four core modules are functional, together with user authentication and account management.

## What's new

### Authentication & user management

- Login screen shown every time VASSL opens — username and password required
- First-launch setup creates the administrator account (no additional software needed)
- Administrator accounts have full access to every module
- Administrators can create accounts for other staff members with per-module permissions:
  - Inventory, Price Book, Quotations access toggles
  - Allow Delete and Allow Price Edit flags
- Administrators can reset passwords, edit permissions, deactivate, and reactivate accounts
- Log out at any time with **Cmd/Ctrl+Shift+L** or via Settings → Security
- Passwords are stored securely using Argon2 hashing — plain text is never recorded

### Inventory
- Product catalogue with SKU, name, category, unit, model number, part number, duty %, and minimum stock level
- Optional long-form product description field
- Preferred supplier assignment per product
- Stock entries for tracking deliveries and acquisitions — types: `Restock`, `Project`, `Adjustment`
- Stock status indicators: Healthy (green) · Low (amber, within 20 % of min) · Critical (red, at or below min)
- Real-time filter search by name, SKU, and category
- Context menu on product rows: Edit, Price History, Add Price Entry

### Price Book
- Cost price, duty % (from product), markup %, and selling price per product
- Currency toggle per entry: USD or JMD
- Prices displayed in the currency they were entered in ($ for USD, J$ for JMD) — no automatic conversion in the table
- Selling price formula: `Cost × (1 + Duty%/100) × (1 + Markup%/100)`
- Full price history — every change recorded with a date
- Real-time filter search
- Price history context menu on rows

### Quotations
- Project-based quotation management with full client details (name, address, attention, telephone)
- Auto-generated reference numbers in `VASSL-YYYY-NNNN` format
- Quotation fields: Exchange Rate (JMD/USD), Discount %, GCT % (Jamaica's General Consumption Tax), Validity (days), Notes
- Exchange rate defaults to the value in Settings → Price Book — no need to type it on every quotation
- JMD total in the quotation footer uses the quotation's stored rate (or current Settings rate as fallback)
- **Edit quotation** — Draft status quotations can be edited after creation: project, exchange rate, discount, GCT, validity, and notes
- Draft → Sent → Approved / Rejected status workflow
- Line items: product picker auto-fills description and unit price from Price Book; per-line discount %
- Automatic stock entry creation in Inventory on quotation approval

### Suppliers
- Supplier contact management: name, contact person, email, phone, notes
- Real-time filter search
- Preferred supplier assignments from Inventory product edit form
- Edit existing suppliers in place

### Settings
- **Appearance** — font family picker, font size control (10–24 pt), OS dark/light theme sync with manual override
- **Security** — change your own password from within the app
- **Admin** (administrators only) — full user management panel
- **Price Book** — USD → JMD exchange rate setting, used as default for new quotations

### App shell
- Native macOS menu bar (VASSL, File, Edit, View, Window, Help)
- Custom Windows title bar with caption buttons (minimize, maximize, close)
- Global Search (`Cmd/Ctrl+Shift+F`) searches products, suppliers, and quotations across all modules
- Module search bar focus (`Cmd/Ctrl+F`) filters the active module's list
- `Cmd/Ctrl + =` / `Cmd/Ctrl + -` keyboard shortcuts for font size
- Log out shortcut: `Cmd/Ctrl+Shift+L`
- Audit Log (`Cmd/Ctrl+Shift+A`) with full change history
- Scrollable lists with draggable scrollbar thumb in all modules
- Tooltips on all sidebar buttons and toolbar action buttons showing keyboard shortcuts

### Platform
- macOS (Apple Silicon, Intel, Universal Binary)
- Windows x86_64 — custom borderless title bar to match app theme
- GitHub Actions release workflow for automated macOS and Windows builds
- Release channel inferred from version: `0.x.y` → Dev · `x.y.z-preview` → Preview · `x.y.z` → Stable
- 7-day rotating log files in OS local data directory

## Known limitations

- No data export (CSV / Excel) yet
- No cloud sync or backup
- App is unsigned — macOS Gatekeeper requires right-click → Open on first launch
