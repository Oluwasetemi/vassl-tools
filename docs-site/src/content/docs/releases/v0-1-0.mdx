This is the initial development release of VASSL. All core modules are functional.

## What's new

### Inventory
- Product catalogue with SKU, name, category, unit, and minimum stock level
- Stock entries for tracking deliveries and acquisitions
- Low-stock status indicators (green / amber / red)
- Real-time filter search
- Product description field
- Preferred supplier assignment per product

### Price Book
- Cost price, duty cost, markup %, and selling price per product
- Full price history — every change is recorded
- Auto-calculation of selling price from cost + duty + markup
- Price history context menu
- Real-time filter search

### Quotations
- Project-based quotation management
- Draft → Sent → Approved / Rejected workflow
- Line items sourced from the price book
- Automatic stock entry creation on quotation approval

### Suppliers
- Supplier contact management (name, contact person, email, phone)
- Real-time filter search
- Preferred supplier assignments from Inventory

### App shell
- Native macOS menu bar (VASSL, File, Edit, View, Window, Help)
- Global Search (`Cmd+Shift+F`) across all modules
- Command Palette (`Cmd+F`) per module
- Font family picker and font size controls in Settings
- OS dark/light mode automatic theme switching
- Audit Log with full change history
- `Cmd/Ctrl + =/-` keyboard shortcuts for font size

### Platform
- macOS (Apple Silicon, Intel, Universal Binary)
- Windows (x86_64)
- GitHub Actions release workflow for automated builds
- Release channel inferred from version number (`0.x.y` → Dev, `x.y.z-preview` → Preview, `x.y.z` → Stable)

## Known limitations

- No data export (CSV / Excel) yet
- No multi-user support — single database file per machine
- No cloud sync or backup
- App is unsigned — macOS Gatekeeper requires manual bypass on first launch
