---
title: Global Search
description: Searching across all VASSL modules at once.
---

Global Search lets you find records across all modules from a single query, without switching between panels.

Open with **Cmd+Shift+F** (macOS) / **Ctrl+Shift+F** (Windows).

## What it searches

Global Search queries across:

| Module | Searched fields |
|---|---|
| Inventory | Product name, SKU |
| Suppliers | Supplier name, contact person |
| Quotations | Project name, client name |

## Using Global Search

1. Press **Cmd+Shift+F** to open the overlay.
2. Type your query — results appear as you type.
3. Use **Up / Down** arrows to navigate results.
4. Press **Enter** to open the selected record in its module.
5. Press **Escape** to close without navigating.

## Command Palette

The **Command Palette** (`Cmd+F`) is a module-level search — it searches only within the currently active module. Use it when you know which module your record is in.

| Tool | Shortcut | Scope |
|---|---|---|
| Command Palette | `Cmd+F` | Current module only |
| Global Search | `Cmd+Shift+F` | All modules |
