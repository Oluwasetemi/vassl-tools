title: Keyboard Shortcuts
description: Complete keyboard shortcut reference for VASSL.
---

`secondary` maps to **Cmd** on macOS and **Ctrl** on Windows/Linux.

## Navigation

| Shortcut | Action |
| --- | --- |
| `secondary-1` | Open Inventory |
| `secondary-2` | Open Quotations |
| `secondary-3` | Open Price Book |
| `secondary-4` | Open Suppliers |
| `secondary-n` | New Record (context-sensitive) |
| `secondary-,` | Open Settings |
| `secondary-q` | Quit VASSL |

## Search

| Shortcut | Action |
| --- | --- |
| `secondary-f` | Open Command Palette (module search) |
| `secondary-shift-f` | Open Global Search (across all modules) |

## View

| Shortcut | Action |
| --- | --- |
| `secondary-=` | Increase font size |
| `secondary-shift-=` | Increase font size (alternative) |
| `secondary--` | Decrease font size |

## Overlays & modals

| Shortcut | Action |
| --- | --- |
| `Escape` | Close active overlay / modal |

## Window (macOS)

| Shortcut | Action |
| --- | --- |
| `Cmd-m` | Minimize window |
| `Cmd-shift-a` | Open Audit Log |

## Text input

These shortcuts work whenever a text field is focused.

| Shortcut | Action |
| --- | --- |
| `Backspace` | Delete character before cursor |
| `Delete` | Delete character after cursor |
| `Left / Right` | Move cursor |
| `Shift-Left / Shift-Right` | Extend selection |
| `secondary-a` | Select all |
| `Home / End` | Jump to start / end |
| `secondary-v` | Paste |
| `secondary-c` | Copy |
| `secondary-x` | Cut |
| `Tab / Shift-Tab` | Next / previous field |
| `Ctrl-Cmd-Space` | Show character palette (macOS) |

## Form navigation

| Shortcut | Context | Action |
| --- | --- |---|
| `Tab` | Product form | Next field |
| `Shift-Tab` | Product form | Previous field |
| `Escape` | Product form | Cancel / close form |
| `Tab` | Stock entry form | Next field |
| `Tab` | Price entry form | Next field |
| `Tab` | Supplier form | Next field |
| `Tab` | Project form | Next field |
| `Tab` | Line item form | Next field |
| `Escape` | Any form | Cancel |

## Command Palette

| Shortcut | Action |
| --- | --- |
| `Down` | Next result |
| `Up` | Previous result |
| `Enter` | Confirm selection |

## Global Search

| Shortcut | Action |
| --- | --- |
| `Down` | Next result |
| `Up` | Previous result |
| `Enter` | Open selected record |
